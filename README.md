# KMS

website khusus member KMS

## Setup
### Requirements:
- Go > 1.10
- Node.js >= 18
- PostgreSQL

### Dev

**Frontend:**
```bash
npm install # to install required dependencies
npm run dev # to run dev
npm run build # to build production
```

**Backend:**
```bash
go mod download # to get required packages
go run . # to run dev
go build . # to build production
```
